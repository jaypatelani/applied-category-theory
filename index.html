Introduction

Introduction

Hi! Welcome to the GitBook-version of the Applied Category Theory Course. The full course exists on the Azimuth Forum and this is a collection of the lectures that John Baez posted in the forum, in a GitBook-format.

To begin, you can download a copy of the text:

  • Brendan Fong and David Spivak, Seven Sketches in Compositionality: An Invitation to Applied Category Theory.

Then start reading the lectures.

If you want to actively participate in the course, you can start with John Baez's welcome message. There you'll find information on registering with the forum and participating in the discussion.

If you think you've found a mistake in the book, please report it here:

  • Seven Sketches book: Typos, comments, questions, and suggestions.

If it's really a mistake, Fong and Spivak will fix it!
